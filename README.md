# Образовательная онлайн-платформа

Веб-платформа для размещения и прохождения онлайн-курсов, разработанная
в рамках курсового проекта за 4 курс.

Проект представляет собой урезанный аналог образовательной платформы Skillspace.

---

## Функциональность

### Пользователи и доступ
- Ролевая модель: **преподаватель**
- Регистрация и авторизация пользователей
- Аутентификация на основе JWT

### Курсы и учебный контент
- Создание и управление курсами
- Создание учебных уроков
- Написание лекционного материала с использованием **EditorJS**
- Создание тестовых заданий
- Создание и проведение вебинаров

---

## Стек технологий

### Frontend:
- Next.js
- TypeScript
- TailwindCSS
- EditorJS
- Redux Toolkit
- Axios
- React-hook-form

### Backend:
- NestJS
- PrismaORM

### DB:
- PostgreSQL

---

## Запуск приложения

### Frontend:
```bash
- cd zskills-frontend
- npm i
- npm run dev
```

### Backend:
```bash
- cd zskills-backend
- npm i
- npm run start:dev
```
